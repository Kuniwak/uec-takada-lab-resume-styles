h1. "AnZen-Lab (UEC, Tokyo)":http://www.az.inf.uec.ac.jp/  Resume Styles(TeX+BibTeX)

LaTeX style formats, samples and utilities. And using BibTeX to management referencies.
These file encoding is EUC-JP.

h2. Requirements

Makefile uses above software.

* platex
* bibtex
* dvipdfmx

So if you want to use "Make" command, you have to install there software before using.

h2. HowTo

# Modify info.def, body.tex, bibtex.bib
# Execute "make"  on Terminal.

You may have to change current directory to a directory including "Makefile".
